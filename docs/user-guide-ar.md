# دليل استخدام Cost Control System

## 1. فكرة النظام
النظام مخصص لمراقبة المخزون والتكاليف والمبيعات والمصروفات للفروع، مع اعتبار كل فرع مخزنًا مستقلًا، ومع خصم الريسبي مباشرة من خامات الفرع عند البيع أو الضيافة أو الإعدام.

## 2. ترتيب التشغيل الصحيح
1. إدخال الوحدات والمجموعات.
2. إدخال الخامات والمنتجات والفروع والموردين والخزائن.
3. تجهيز الريسبيات العامة أو الخاصة بفروع معينة.
4. إدخال رصيد أول المدة.
5. تسجيل المشتريات والتحويلات.
6. بدء تسجيل المبيعات والضيافة والعمليات المخزنية.
7. تسجيل مصروفات التشغيل وسداد الموردين.
8. متابعة التقارير المالية والتشغيلية.

## 3. الشاشات الأساسية
- `الوحدات / المجموعات / الخامات / المنتجات`: التأسيس الأساسي للنظام.
- `الريسبيات`: تحديد مكونات كل صنف مع إمكانية تخصيص الريسبي حسب الفرع.
- `الفروع`: تعريف الفروع باعتبارها مخازن تشغيلية مستقلة.
- `الموردون`: تعريف الموردين وكشف حسابهم وسدادهم.
- `المشتريات`: إدخال فواتير الشراء وتحديث متوسط التكلفة.
- `المبيعات`: إدخال البيع والضيافة والـ Void مع خصم الريسبي من المخزون.
- `العمليات المخزنية`: رصيد أول المدة، تحويلات، جرد، هالك، مرتجعات.
- `الخزائن والبنوك`: متابعة النقدية ووسائل التحصيل والصرف.
- `مصروفات التشغيل`: تحميل المصروفات على الفروع والمنتجات.
- `التقارير المالية`: دفتر اليومية، ميزان المراجعة، الميزانية، التدفقات النقدية.

## 4. الدورة اليومية المقترحة
1. مراجعة المشتريات أو التحويلات الواردة.
2. تسجيل المبيعات اليومية لكل فرع.
3. تسجيل الضيافة والـ Void إن وجدت.
4. تسجيل المصروفات التشغيلية وسداد الموردين.
5. متابعة الداشبورد والتنبيهات وتقارير الانحراف.

## 5. أهم التقارير
- الداشبورد التنفيذي.
- أرصدة المخزون وكارت الصنف.
- تكلفة المنتجات وملخص التكاليف.
- تقرير الريسبيات حسب الفرع.
- انحراف المخزون والجرد.
- قائمة تكلفة البضاعة المباعة.
- قائمة الدخل للفروع.
- دفتر اليومية وميزان المراجعة والميزانية العمومية والتدفقات النقدية.

## 6. ملاحظات تشغيلية مهمة
- لا تبدأ البيع قبل إدخال رصيد أول المدة أو المشتريات.
- أي صنف يباع يجب أن يكون له ريسبي صحيح.
- يفضل مراجعة متوسط التكلفة بعد أول أسبوع تشغيل.
- يفضل أخذ نسخة احتياطية قبل أي تعديل كبير أو استيراد Excel.

## 7. بيانات الدخول الافتراضية
- اسم المستخدم: `admin`
- كلمة المرور: `123456`

يوصى بتغيير كلمة المرور فور أول تشغيل.
